# PromptDocs

An interactive, gamified platform for mastering prompt engineering through tutorials, challenges, and comprehensive learning resources.

## 🚀 Features

- **Interactive Learning**: Step-by-step tutorials and hands-on exercises
- **Gamified Experience**: Earn XP, unlock achievements, and track progress
- **Comprehensive Topics**: From zero-shot to advanced prompt engineering techniques
- **Modern UI**: Built with React, featuring a sleek dark theme with neural network animations
- **Responsive Design**: Works seamlessly on desktop and mobile devices

## 🛠️ Tech Stack

- **Frontend**: React 18, Vite
- **Styling**: Custom CSS with CSS Grid and Flexbox
- **Icons**: Lucide React
- **Fonts**: Bebas Neue, DM Sans, Space Mono

## 📦 Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/promptdocs.git
cd promptdocs
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open [http://localhost:5173](http://localhost:5173) in your browser.

## 🏗️ Build for Production

```bash
npm run build
```
